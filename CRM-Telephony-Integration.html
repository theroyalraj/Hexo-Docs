    Proxy Call

    A proxy call is when we connect the Salesken dialer server with the agent phone and then call the customer.
    Which means the user dials a number and first a call is initiated from caller id to user’s own number and after users picks up the call, there is a call conference between user’s number, caller id and the dialed number.
